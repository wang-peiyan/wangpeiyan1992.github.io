检查本机是否安装Node

  • 执行该命令 npm node -v
  • 如果已安装进行会显示版本号
  • 如果未安装请先安装node

检查本机是否安装Git

  • npm git -v
  • 如果已安装进行会显示版本号
  • 如果未安装请先安装git
  • mac版下载的软件提示无法打开时 按control＋点击打开便可

GitHib账号如没有先注册

打开github网站登录

  • 点击右上角，new repository
  • 填写仓库信息
  • 点击setting
  • 点击
  • 不作任何修改，点击
  • 不做任何修改，点击publish page
  • 此时在浏览器中输http:lilianghong.github.io/ 应该可以打开博客

首先设置你的用户名密码

  • git config –global user.email “wpyphp@163.com”
  • git confit –global user.name “wangpeiyan1992”

在命令行下，生称sshkey

  • ssh-keygen -t rsa -C “wpyphp@163.com”
  • ssh-keygen -t rsa -C “wpyphp@163.com” -f ~/.ssh/github 生成指定名字的key 此名字为github
  • 生成密钥后到cd ~/.ssh 下打开github.pub 全选复制打开github－setting－add sshkey 将刚才复制的内容粘贴在此处 保存

验证一下是否成功

  • ssh -T git@github.com
  • 成功执行下一步 失败请重新设置

安装hexo，执行以下命令

  • npm install -g hexo 注意：mac本要加sudo
  • npm hexo -v #查看版本号

自己新建一个目录，到目录里初始化

  • hexo init wangpeiyan1992.github.io

找到该目录下config.yml文件打开添加以下代码

  • 注意根据自己本机安装有几个项目和密钥 对应几个配置 

    host github.com
    user git 
    hostname github.com
    port 22
    identityfile ~ /.ssh/github

host git.emao.net
    user git 
    hostname git.emao.net
    port 22
    identityfile ~/.ssh/emao

生成静态页面

  • hexo generate

本地启动服务

  • hexo server

此时浏览器输入以下地址就可以看到效果

  • http://localhost:4000

创建一篇博客

  • 新建 hexo new “my first blog” 注意：服务不能关闭 可以新打开一个终端
  • 此时到io - source - _posts 下找到新建的文件

  • 打开文件看到以下内容

    ---
title: { { title } } postName #文章页面上的显示名称，可以任意修改，不会出现在URL中
date: { { date } } #文章生成时间，一般不改，当然也可以任意修改
categories: #文章分类目录，可以为空，注意:后面有个空格
tags: #文章标签，可空，多标签请用格式[tag1,tag2,tag3]，注意:后面有个空格
---
这里开始使用markdown格式输入你的正文。
  • 用喜欢的编辑器编辑即可

生成并发布

方法一

  • 生成 hexo generate
  • 发布 hexo deploy

方法二

  • hexo deploy －g

总结常用命令

  • hexo new “postName” 新建文章
  • hexo new page “pageName“ 新建页面
  • hexo generate 生成页面
  • hexo server 开启服务
  • hexo deploy 发布文章

常用符合命令

  • hexo deploy -g 生成并发布
  • hexo server -g 启动服务并生成页面

简写命令

  • hexo n == hexo new
  • hexo g == hexo generate
  • hexo s == hexo server
  • hexo d == hexo deploy

主题安装

  • git clone https://github.com/heroicyang/hexo-theme-modernist.git themes/modernist

  • 安装完成后，打开hexo|_config.yml 修改主题为modernist
    theme: modernist

  • 打开hexo|themes|modernist目录，打开_config.yml

    menu: #配置页头显示哪些菜单
#  Home: /

  Reading: /reading
  About: /about
#  Guestbook: /about

excerpt_link: Read More #摘要链接文字
archive_yearly: false #按年存档

widgets: #配置页脚显示哪些小挂件

#  - tag
  - tagcloud
  - recent_posts
